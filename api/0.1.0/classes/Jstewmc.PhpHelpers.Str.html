    Returns a random string of $length that follows the charset $rules

    password(integer $length = 8, array<mixed,integer> $rules = array('lower' => 1, 'upper' => 1, 'number' => 1, 'symbol' => 1)) : string
    Static
    Oftetimes, standards (like PCI) require passwords with one upper-case letter, one lower-case letter, one number, and one symbol. I can do that. For example: $rules = ['upper' => 12]; $a = Str::password(12, $rules); $rules = ['lower' => 6, 'upper' => 6]; $b = Str::password(12, $rules); $rules = ['lower' => 4, 'upper' => 4, 'number' => 4]; $c = Str::password(12, $rules); echo $a; // example 'KNVHYUIDGVDS' echo $b; // example 'jNhGFkLekOfV' echo $c; // example 'la9Uh7BH4Bc3'
    since 0.1.0

    Parameters

    $length

    integer

    the length of the password (optional; if omitted,

    defaults to 8)

    $rules

    array<mixed,integer>

    an array of character counts indexed by charset name

    (possible charset names are 'lower', 'upper', 'number', 'alpha', and 'symbol')
(optional; if omitted, defaults to ['lower' => 1, 'upper' => 1, 'number' => 1,
'symbol' => 1])

    Exceptions

    \BadMethodCallException if $rules or $length is omitted
    \InvalidArgumentException if $rules is not an array
    \InvalidArgumentException if $length is not an integer
    \InvalidArgumentException if a key in $rules is not a valid charset name
    \InvalidArgumentException if a value in $rules is not an integer
    \InvalidArgumentException if the number of required characters (as defined in the $rules array) exceeds the $length

    Returns

    stringthe password

    Splits a string on the first alpha character

    splitOnFirstAlpha(string $string) : array<mixed,string>
    Static
    I'll return an array with two parts. The first element is the string part before the first alpha character, and the second part is everything after and including the first alpha character. For example: Str::splitOnFirstAlpha("123"); // returns ["123"] Str::splitOnFirstAlpha("abc"); // returns ["", "abc"] Str::splitOnFirstAlpha("123 abc"); // returns ["123", "abc"] Str::splitOnFirstAlpha("1 2 3 abc"); // returns ["1 2 3 4", "abc"]

    Converts a php.ini-like byte notation shorthand to a number of bytes

    strtobytes(string $string) : integer | float
    Static
    In the php.ini configuration file, byte values are sote in shorthand notation (e.g., "8M"). PHP's native ini_get() function will return the exact string stored in php.ini and not its integer equivalent. I will return the integer equivalent. For example: Str::strtobytes('1K'); // returns 1024 Str::strtobytes('1M'); // returns 1048576 Str::strtobytes('1G'); // returns 1073741824
    since 0.1.0
    see ini_get() man page

    string

    the string to convert

    Exceptions

    \BadMethodCallException if $string is null
    \InvalidArgumentException if $string is not a string
    \InvalidArgumentException if $string does not end in 'k', 'm', or 'g'

    integerfloatthe number of bytes
