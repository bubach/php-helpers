    Copies files or directory to the filesystem

    copy(string $source, string $destination, integer $mode = 511) : boolean
    Static
    PHP's native copy() function only copies files, not directories. I will recursively copy a directory and all of its files and sub-directories. If the $destination exists, I will overwrite any existing files with the corresponding file in the $source directory. If $destination does not exist, and $mode is set to false I will throw an InvalidArgumentException. If $mode is an integer (or omitted), I attempt to create the destination directory. I will recursively create destination directories as needed. To copy a file, use PHP's native copy() method.
    since 0.1.0
    see Felix King's answer to "Copy entire contents of a directory to another using php" on StackOverflow

    Parameters

    $source

    string

    the source directory path

    $destination

    string

    the destination directory path

    $mode

    integer

    the mode of the destination directory as an octal number with a leading zero (ignored on Windows) (optional; if omitted, defaults to 0777, the widest possible access) (set to false to throw an exception if the destination directory does not exist)

    Exceptions

    \BadMethodCallException if $source, $destination, or $mode is null
    \InvalidArgumentException if $source is not a string
    \InvalidArgumentException if $destination is not a string
    \InvalidArgumentException if $mode is not an integer or false
    \InvalidArgumentException if $source does not exist or is not a directory
    \InvalidArgumentException if $source is not readable
    \InvalidArgumentException if $destination does not exist or it could not be created successfully
    \InvalidArgumentException if $destination is not writeable

    Returns

    booleantrue if successful

    Deletes a non-empty directory and its sub-directories

    remove(string $directory, string $container) : boolean
    Static
    PHP's native rmdir() function requires the directory to be empty. I'll recursively delete a directory's files and sub-directories. BE CAREFUL! Use the $container argument to be safe.
    since 0.1.0
    see donald123's answer to "Remove all files, folders, and their subfolders with php" on StackOverflow
    see rmdir() man page

    Parameters

    $directory

    string

    the path of the directory to remove

    $container

    string

    an ancestor directory of $directory

    Exceptions

    \BadMethodCallException if $directory or $container is null
    \InvalidArgumentException if $directory is not a string
    \InvalidArgumentException if $container is not a string
    \InvalidArgumentException if $directory is not a valid directory path
    \InvalidArgumentException if $directory is not writeable
    \InvalidArgumentException if $directory is not contained in $container

    Returns

    booleantrue if success
